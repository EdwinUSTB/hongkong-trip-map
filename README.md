# Hong Kong Trip Map

这是 7月14日至17日香港旅行地图。

## 使用方式

- 网站入口：启用 GitHub Pages 后访问 `https://edwinustb.github.io/hongkong-trip-map/`
- 地图页面：`index.html`
- 可导入 Google My Maps / Google Earth 的点位文件：`hongkong_2026-07-15_points.kml`

## GitHub Pages 设置

如果页面还不能打开，请在 GitHub 仓库页面进入：

`Settings` -> `Pages` -> `Build and deployment` -> `Deploy from a branch`

选择：

- Branch: `main`
- Folder: `/root`

保存后等待 1-3 分钟，页面会发布到：

`https://edwinustb.github.io/hongkong-trip-map/`
